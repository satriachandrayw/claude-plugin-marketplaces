# Skills

Custom plugins and marketplace for Claude Code skills.

## Plugins

### Product Manager
Product Manager skill for creating well-structured GitHub issues from feature requests and bug reports.

### Agent Knowledge
Agent Knowledge Base - Captures and organizes learnings from external AI agents into a structured knowledge base.

## Installation

In Claude Code, run:
```
/plugin add https://github.com/satriachandrayw/skills
```

## Usage

After installation, the skills will be available in your Claude Code session.
