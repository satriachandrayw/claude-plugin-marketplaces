# Skills

Custom plugins and marketplace for Claude Code skills.

## Plugins

### Product Manager
Product Manager skill for creating well-structured GitHub issues from feature requests and bug reports.

### Agent Knowledge
Agent Knowledge Base - Captures and organizes learnings from external AI agents (Codex, etc.) into a structured knowledge base.

### Bug Hunter
Random bug hunter that selects files weighted by recency, analyzes with context, and creates GitHub issues for approved findings.

### Bug Fixer
TDD-based bug fixer that reads GitHub issues, implements fixes, and creates PRs with severity-based batching (max 5 per batch).

### Create PR
Post-task PR creation with git context analysis and issue linking.

## Installation

In Claude Code, run:
```
/plugin add https://github.com/satriachandrayw/skills
```

## Usage

After installation, the skills will be available in your Claude Code session.
